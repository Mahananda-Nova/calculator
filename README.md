# Calculator Project

## Overview

A simple calculator web application built using HTML, CSS, and JavaScript. The calculator performs basic arithmetic operations and provides a clean user interface.

## Technologies Used

### HTML

* Creates the structure of the calculator.
* Contains buttons and display screen.

### CSS

* Styles the calculator layout.
* Improves appearance and responsiveness.
* Designs buttons and display.

### JavaScript

* Handles button clicks.
* Performs calculations.
* Clears and updates the display.

## Features

* Addition (+)
* Subtraction (-)
* Multiplication (*)
* Division (/)
* Clear Button (C)
* Delete Button (DEL)
* Real-time Result Calculation

## How to Run

1. Download the project files.
2. Open `index.html` in any web browser.
3. Start performing calculations.
